# Apptainer example

This is a simple example of a workflow downloading CIF files from the Crystallography Open Database and performing GPAW calculations in an Apptainer container.

You can execute the workflow as follows:

```bash
stepup boot
```

Note that the CIF downloads are not always successful, so you may have to run the workflow a few times before it completes successfully.
