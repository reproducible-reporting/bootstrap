# Matplotlib example

This example consists of two scripts:

- `generate.py` generates a Fibonacci sequence and saves it to a file.
- `plot.py` reads the Fibonacci sequence from the file and plots it.

The `plan.py` file defines a StepUp workflow that runs these two scripts in sequence.

You can execute the workflow as follows:

```bash
stepup boot
```
